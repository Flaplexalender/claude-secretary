MUTATION APPLICATION ENGINE
============================

evolve_prompt(prompt: str, failure_pattern: FailurePattern, num_mutations: int = 2)
  → (evolved_prompt: str, mutations_applied: list[PromptMutation])

Algorithm:
  1. Look up applicable mutation types for the failure pattern
  2. Find all rules matching those types and the failure pattern
  3. Apply top N rules sequentially (default: 2, max: 3)
  4. Return evolved prompt + audit trail with before/after segments

Features:
  - Zero cost: pattern matching only, no LLM calls
  - Reversible: full mutation history tracked
  - Composable: multiple mutations can be chained
  - Observable: human-readable reports generated
